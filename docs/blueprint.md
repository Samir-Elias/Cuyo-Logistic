# **App Name**: Cuyo Logistics

## Core Features:

- Hero Section: Eye-catching landing section with a background image, company logo, strong title, and a clear call-to-action button that links to the contact form.
- About Us Section: Details the company's history, background since 2000, and operational scope across Mendoza, Buenos Aires, and Rosario.
- Services Section: Presents the three core services—Liquid Transport, Containers, and Modular Housing (Bunka Modules)—using a card-based layout with icons, titles, and concise descriptions.
- Contact Form: A clean and responsive form for inquiries, including fields for Name/Company, Email, Phone, Service Type (dropdown: Liquids, Containers, Modules, General), and Message. Form submissions are simulated via a placeholder webhook.
- Social Media Links: Links to social media platforms (LinkedIn, Facebook) and contact information (email, phone).

## Style Guidelines:

- Background color: Dark gray (#333333) to provide a professional and solid base.
- Primary color: Blue (#3498db) to inspire trust and confidence.
- Accent color: A lighter cyan (#A0E7E5), to create good contrast for interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern and clean appearance.
- Use vector icons from Heroicons or Lucide to represent services.
- Sticky header navigation for easy access to all sections.
- Subtle transition animations for a smooth user experience.
